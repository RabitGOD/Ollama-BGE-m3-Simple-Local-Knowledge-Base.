Ollama BGE-m3 简易本地知识库

使用BGE-m3获取文本嵌入向，FAISS索引搜索最相似的几个文档。后转换再归一化计算余弦相似度1 - cosine(u, v)。（余弦相似度阈值设置了0.5）

使用本地知识库相对隐私一些，虽简单暴力，但精度一般。

需要去OLLAMA官网拉取BGE-M3模型即可
